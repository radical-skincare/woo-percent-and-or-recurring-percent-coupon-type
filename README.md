# Woo Percent and or Recurring Percent Coupon Type

## Description

WordPress Plugin to create a WooCommerce Perfect and/or Recurring Coupon Type

## Requirements

* [WordPress](http://wordpress.org/) CMS
* [WooCommerce](https://wordpress.org/plugins/woocommerce/) WordPress plugin
* [Vitalibis Partners](https://partners.vitalibis.com/login) Account
* [GigfilliateWP](https://github.com/radical-skincare/gigfilliate-wp) WordPress plugin

## Installation

How to install the WordPress plugin and get it working.

1. Upload `woo-percent-and-or-recurring-percent-coupon-type` to the `/wp-content/plugins/` directory. Ensure sure plugin folder name is "woo-percent-and-or-recurring-percent-coupon-type"
2. Go to Admin Plugins `/wp-admin/plugins.php` and Activate the "Woo Percent and or Recurring Percent Coupon Type" plugin
